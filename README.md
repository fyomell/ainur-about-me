# Ainur Rofik • About Me (Next.js + TypeScript + Tailwind)

## How to run
1) Install Node.js (LTS)  
2) Open this folder in Terminal  
3) Install dependencies:
```bash
npm install
```
4) Start dev server:
```bash
npm run dev
```
Open: http://localhost:3000

## Edit content
All profile text is in:
- `lib/profile.ts`
